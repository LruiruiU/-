# 条件语句和逻辑运算符如何使用

* 条件语句，或称条件句，可以让你根据某些条件的真假来控制程序的流程
* 先来从比较算子开始了解

```
| 运算符 |  名称  |         用法           |
|-------|-------|------------------------|
|   ==  |  等于 |   print(3 == 4) # False |
|-------|-------|------------------------|
|   !=  | 不等于 |  print(3 != 4) # True  |
|-------|-------|------------------------|
|   >   |  大于 |   print(3 > 4) # False  |
|-------|-------|------------------------|
|   <   |  小于 |   print(3 < 4) # True   |
|-------|-------|------------------------|
|   >=  |大于等于|  print(3 >= 4) # False |
|-------|-------|------------------------|
|   <=  |小于等于|  print(3 <= 4) # True  |
|-------|-------|------------------------|
```

# 在Python中最基本的条件语句是：

```
if condition:
    pass # 如果条件为真，代码执行，反之不执行
```

* if 语句以 if 关键词开头
* 条件是一个表达式，其值为真(True)或假(False)，并以冒号结尾
* 语句的主题构成一个**代码块**，即属于同一组的语句
* 在Python中，缩进的层级定义了代码块
* 在上面的示例中，语句的主体里包含了一个语句
* 当该语句被执行时，并不会发生任何操作，这是一个特殊的关键字，可用于作为未来代码的占位符，尤其是在不允许存在空代码块的情况下非常有用
* 当评估条件为真(True)时，语句主体内的代码才会执行：

```
age = 19

if age >= 18:
    print('你已成年') #输出 你已成年
--------------------------------
age = 16

if age >= 18:
    print('你已成年') #不输出任何东西
```

* 要注意缩进，其它语言使用大括号来定义代码块，缩进仅仅作为提升可读性
* 但Python中，代码块是通过缩进决定的：

```
age = 18

if age >= 18:
print('You are an adult') #这样就会报错
```

* 虽然你可以使用任意数量的空格(只要保持一致)来确定每个缩进，但Python风格指南中建议使用四个空格

# 如果你想打印不等于条件的结果的话，就该使用 clase 了

* **clse** ：当条件为假时运行该语句

```
if condition:
   pass # 条件为真执行
else:
   pass # 条件为假执行
```

* 例如：

```
age = 16

if age >= 18:
    print('你已成年')
else:
    print('你还未成年') # 输出 你还未成年
```

* 注意，不要在这中间放置任何语句，否则就会报错

# 有些情况下你可能需要考虑多种情况，在这种情况下，Python允许使用多个关键字来扩展你的 if 语句：

```
if condition1:
   pass # 当 condition1 为真时执行
elif condition2:
   pass # 如果 condition1 为假但 condition2 为真时执行
else:
   pass # 如果所有条件为假时执行
```

‍

* 示例：

```
age = 12

if age >= 18:
    print('你已城建')
elif age >= 13:
    print('你还是青少年')
else:
    print('小孩别闹') # 输出 小孩别闹
```

* 你可以设置任意数量的 else 前提是你真的有这么多需要判断的
