# 列表的常用的使用方法

## 首先是 append()

* append() 用于将元素添加至列表末尾：

```
numbers = [1, 2, 3, 4, 5]
numbers.append(6)

print(numbers) #输出 [1, 2, 3, 4, 5, 6]
```

* 如果你想在列表末尾嵌套一个列表
* 那么也可以用它：

```
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.append(even_numbers)
print(numbers) # [1, 2, 3, 4, 5, [6, 8, 10]]
```

* 注意，这个方法是将 even\_numbers 嵌套进了 numbers 里

## 如果你不想嵌套而是想添加到列表末尾的话，就可以使用 extend()

```
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.extend(even_numbers)
print(numbers) #输出 [1, 2, 3, 4, 5, 6, 8, 10]
```

## 如果你想将元素插入到列表中的特定索引处，那么可以使用 insert()

* 该方法接受两个参数：你想插入新项的索引和你想插入的项

```
numbers = [1, 2, 3, 4, 5]
numbers.insert(2, 2.5) #指定索引位与需要插入的元素

print(numbers) #输出 [1, 2, 2.5, 3, 4, 5]
```

* 这样就把 2.5 插入到了 2 索引处

## 如果你想从列表中移除某个元素，那么可以使用 remove()

```
numbers = [1, 2, 3, 4, 5]
numbers.remove(3) #指定需要删除的元素

print(numbers) #输出 [1, 2, 4, 5]
```

* 注意，若列表中有重复的元素使用这个方法移除的话
* 该方法只会移除首次出现的元素，并非全部

```
numbers = [1, 2, 3, 4, 5, 3]
numbers.remove(3)

print(numbers) #输出 [1, 2, 4, 5, 3]
```

## 如果需要移除列表中指定索引处的元素，可以使用 pop()

```
numbers = [1, 2, 3, 4, 5]
numbers.pop(1) #将 2 移除
```

* 如果没有指定索引的话则会默认移除最后的元素

```
numbers = [1, 2, 3, 4, 5]
numbers.pop() #将 5 移除
```

## 如果你需要清空列表，那么可以使用 clear()

```
numbers = [1, 2, 3, 4, 5]
numbers.clear()

print(numbers) #输出 []
```

## 接下来是 sort()

* 该方法用于将元素按序排列

```
numbers = [19, 2, 35, 1, 67, 41]
numbers.sort()

print(numbers) #输出 [1, 2, 19, 35, 41, 67]
```

## 还有 sorted()

* 该方法并不是修改原有列表
* 该函数适用于任何可迭代对象
* 并返回一个新的排序列表

```
numbers = [19, 2, 35, 1, 67, 41]
sorted_numbers = sorted(numbers)

print(numbers)        #输出 [19, 2, 35, 1, 67, 41]
print(sorted_numbers) #输出 [1, 2, 19, 35, 41, 67]
```

* 注意，可迭代对象是一种特殊类型的对象
* 可以对其进行循环遍历从而访问每个项目

## 还有就是 reverse()

* 该方法用于将列表中的元素反转

```
numbers = [6, 5, 4, 3, 2, 1]
numbers.reverse()

print(numbers) #输出 [1, 2, 3, 4, 5, 6]
```

## 最后就是 index()

* 该方法用于查询某个元素在列表中第一次出现的位置

```
programming_languages = ['Rust', 'Java', 'Python', 'C++']
print(programming_languages.index('Java')) #输出 1
```

* 如果找不到该元素
* 你就喜提一个大大的 值错误 ValueError
