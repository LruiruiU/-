# 变量是什么

在Python中，变量就如同一个带标签的盒子，用来储存和引用不同类型的数据;

# 如何声名变量

* 在Python中声明变量时需要用 **赋值操作符(=)** 为标识符赋值，，不用像JavaScript或者C语言那样使用特殊的关键词语;
* 在Python中只需要左侧写变量名称，中间是赋值运算符(=),右侧写你想要赋予它的值，以下有一个演示：

```
name = 'xiaoming'
age = 25
```

* 在上述示例中 name = 'xiaoming' 意思就是吧 xiaoming 这个**字符串(用来表示你输入的文本，字符串可以用单引号''或双引号""框住，不能不框)**赋值给 name 这个变量，从这行开始除非重新给 name 这个变量赋值，那么 name就一直等于xiaoming;
* 在Python中为变量赋值时有一些规则需要记住:
* 变量名只能以**字母**或者**下划线(\_)**开头，不能以**数字**开头;
* 变量名只能包含**字母**和**下划线;**
* 变量名**区分大小写**，同一个单词大写和小写被判定为两个变量;
* 变量名**不能使用**Python内置模块的关键词，如 if、class、def 等
* 如果**违反**了以上规则，那么Python就会**报错**，如:

```
    6user_name = liling
    ^
SyntaxError: invalid syntax
```

* 现在再来细讲一下Python中变量的一些常见命名规范:

‍

1. 首先，变量名应使用小写字母，单词之间使用下划线分隔，这被称为 **called snake case(蛇形命名法)**：

```
my_name = 'Colored Glaze'
```

1. 接下来是很想重要的一点：**你的变量应使用描述性名称**，就是要让别人一眼就能看出你这个变量定义的是什么，比如 **年的变量命名为age**、**猫的变量名为cat**、**时间的变量名为time，**为什么要这样做？因为你写出来的代码不仅仅是给你自己看的，你要是A = 年、B = 猫、 C = 时间，这样写的话别人看到就要抓狂了，甚至有可能你自己都会不记得这些是什么东西，这已经是业界默认的，**变量名要做到能见此生意(别人看到这个词语就知道是什么意思)**
2. 还有还有就是要避免单字母的变量名，理由和上面的一样

‍

* 在Python中有一种东西叫做 **注释(#)** 注释符号(#)的意思就是让程序运行时忽略注释符号(#)后面所写的内容:

```
my_name = 'Colored Glaze' #我的名字是琉璃（Colored Glaze）
```

1. 因为程序不会运行#后面的东西，所以写中文也没关系的
2. 你可以使用注释来给自己提个醒，或者用来解释代码，以防团队里的人看不懂你写的是什么
3. **要注意一下，就算有注释可以解释你写的代码是什么，你也不能肆无忌惮地写变量名，依然要遵循 见词生意 并遵循其它命名规则才行**

‍
