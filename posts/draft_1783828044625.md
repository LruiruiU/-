# type()和isinstance()函数是如何工作的

上一篇中，学到了Python中常见的数据类型，这一篇来学如何查看变量的类型；

* 要查看变量的类型可以使用以下函数：
* developer和type()

```
#例如
developer = '琉璃'
print(type(developer)) #意思是鉴别变量 developer 的类型并打印
#输出 <class 'str'> 表示这是一个字符串的类型
```

* 如果你没有未函数提供任何参数，那么就会报错；

```
#报错显示
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: type() takes 1 or 3 arguments
```

### 以下是目前学到的数据类型以及输出的类型

```
#整数
my_integer_var = 10
print(type(my_integer))  #输出 <class 'int'>
#浮点数
my_float = 4.5
print(type(my_float))  #输出 <class 'float'>
#字符串
my_string = 'hello'
print(type(my_string))  #输出 <class 'str'>
#布尔值
my_boolean = True
print(type(my_boolean))  #输出 <class 'bool'>
#集合
my_set = {7, 'hello', 8.5}
print(type(my_set))  #输出 <class 'set'>
#词典
my_dictionary = {'name': 'Alice', 'age': 25}
print(type(my_dictionary))  #输出 <class 'dict'>
#元组
my_tuple = (7, 'hello', 8.5)
print(type(my_tuple))  #输出 <class 'tuple'>
#范围
my_range = range(5)
print(type(my_range))  #输出 <class 'range'>
#列表
my_list = [22, 'Hello world', 3.14, True]
print(type(my_list)) #输出 <class 'list'>
#无
my_none = None
print(type(my_none))  #输出 <class 'NoneType'>
```

* 若是程序中需要验证某个变量是否为特点类型才能对其进行操作，这个时候就是它排上用场了；

### isinstance()

```
#这里有一个变量
account_balance = '12'
#如果你想用它做数学计算的话，那么很抱歉
account_balance = '12'

account_balance / 2
#就会报错
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: unsupported operand type(s) for /: 'str' and 'int'
```

* 要判断该变量是否为整数类型就可以使用；

### isinstance()

```
account_balance = '12'

isinstance(account_balance, int) #输出 False
```

* 这个内置函数可以检查某个变量是否匹配特定的数据类型；
* 它接收一个对象和你想进行对比的类型进行对比，然后返回一个布尔值(True/False)；
* 即使是面对多种类型也可以同时检查哦；

```
account_balance = 12
isinstance(account_balance, (int, float)) 
#因为匹配到了 (int,float) 中的 int 类型，所以输出 Ture
```
