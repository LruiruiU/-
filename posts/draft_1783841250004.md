# 字符串是什么，什么是字符串不可变性

* 字符串是一组被单引号或者双引号包裹的文本文字，在某些编程语言中被单引号包裹的字符串和倍双引号包裹的字符串被区别对待，但在Python中，它们是平等的，所以在处理字符串时可以使用这两种方式：

```
str_1 = 'Hello World'
str_2 = "Hello World"
```

* 如果需要使用多行字符串的话可以用三重单引号或双引号：

```
str_3 = '''1
2
3'''

str_4 = """4
5
6"""
```

### 如果使用的字符串包含了双引号或者单引号的话，有两个选择：

* 使用相反的引号，也就是说如果你的字符串里包含了单引号，那么就用双引号包裹，反之亦然：

```
Colored_Glaze = "It's a sunny day"
xiaoming = 'She said, "Hello World!"'
```

* 或者在字符串中用反斜杠(\)转义单引号或双引号,通过这种方法可以使用单引号或双引号来包裹字符串本身:

```
Colored_Glaze = 'It\'s a sunny day'
xiaoming = "She said, \"Hello World!\""
```

* 有的时候，你可能需要检查一个字符串中是否包含一个或者多个字符，为此，Python提供了一个运算符，返回一个布尔值用于表示指定字符串中是否包含该字符：

```
Colored_Glaze = 'Hello World'

print('Hello' in Colored_Glaze) #True
print('He' in Colored_Glaze)    #True
print('llW' in Colored_Glaze)   #False
print('fo' in Colored_Glaze)    #False
print('e' in Colored_Glaze)     #True
```

### 现在了解了解如何 获取字符串的长度并处理字符串中的单个字符：

* 这个过程叫做 **索引**
* 要计算字符串的长度的话，可以使用内置的函数
* **len()**

```
Colored_Glaze = 'Hello World'

print(len(Colored_Glaze)) #11
```

* 字符串中的每一个字符都有被称为索引的位置，索引是以0为基础的，这意味着字符串的第一个字符所以为0，第二个索引为1，以此类推
* 要通过索引访问字符的话，可以使用方括号[]将你想要访问字符的索引包含在内：

```
Colored_Glaze = 'Hello World'

print(Colored_Glaze[0]) #H
print(Colored_Glaze[6]) #W
```

* 负数的索引也是被允许的，字符串中**倒数第一位**字符的索引可以写为-1，倒数第二位就是-2，以此类推：

```
Colored_Glaze = 'Hello World'

print(Colored_Glaze[-1]) #d
print(Colored_Glaze[-7]) #o
#空格也有索引位，如果写入 -6 的话输出的就是空格啦
```

* 许多其他的编程语言将数据类型大致分为原始类型和引用类型
* 原始类型简单且不可变，意味着一旦声明就无法修改
* 引用类型可以包含多个值，并且是可变或者不可变的
* 但Python并没有在这两者之间划清界限
* 相反，所有数据都被视为对象，有些对象不可变，有些对象则可变

### 字符串是Python中不可变的数据类型，这意味着你可以将不同的字符串重新分配给同一变量

```
Colored_Glaze = 'Hi'
Colored_Glaze = 'Hello'

print(Colored_Glaze) #输出Hello
```

* 但是不允许直接修改字符串：

```
Colored_Glaze = 'Hi'
greeting[0] = 'h' 

#TypeError: 'str' object does not support item assignment
#就报错
```
