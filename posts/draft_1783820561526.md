# **Python 中常见的数据类型有哪些**

* 在处理Python变量之前，了解数据的类型是非常重要的，不同的类型储存着不同的数据;
* Python是动态类型的语言，类似于JavaScript，这意味着不需要声明变量的类型，语言会根据你赋予变量的数据类型知道这是一个什么类型的变量;

```
#例如：
name = 'Colored Glaze' #Python知道这是一个字符串类型的变量
age = 19 #Python知道这是一个整数类型的变量
#下面演示一下C语言和Java的变量
string name = 'Colored Glaze' #需要告诉它这是一个字符串类型的变量
int age = 19 #需要告诉它这是一个整数类型的变量
```

* 虽然Python在这方面是很方便，但也可能会出现意想不到的错误，因为Python中类型错误只在程序检测时运行;
* 由于Python是在运行时确定数据的类型的，所以只有在运行时才能发现类型相关的错误;
* 程序运行时Python会**从上到下逐行执行**代码，如果执行到有错误的那行代码Python就会停止执行并报错
* 相比于Python，有些语言会在程序运行前进行编译，这意味着计算机会提前检查代码并准备运行，在这时候这类语言可以在执行前就发现类型错误，只能说各有优缺;
* 可以总结为以下两点：
* 在Python中，类型错误在执行时显现(也就是程序实际运行并执行到错误的代码时);
* 在编译语言(C语言，Java等)中，是在编译步骤中(程序尚未运行时)捕捉类型错误;

### 以下是Python中常用的数据类型：

* 整数(没有小数的整数) **Integer** :

```
my_integer = 19
print(my_integer) #打印出 19
```

* 浮点数(也就是小数) **Float** ：

```
my_float = 6.6
print(my_float) #打印出 6.6
```

* 字符串(一串用单引号或者双引号包裹的文本) **String** :

```
my_string = 'hello'
print(my_string) #打印出 hello
```

* 布尔值(真或者假) **Boolean** **True / False** ：

```
#真：
my_boolean_true = True
print(my_boolean_true) #打印出 True 
#假：
my_boolean_false = False
print(my_boolean_false) #打印出 False
```

### **以上四种统称为 元素**

### **以下是元素的各种 容器**

* 集合(一组无序的唯一元素) **Set** ：

```
my_set = {6.6,'hello',7}
print(my_set) #注意，因为集合是无序的，所以打印时不会按照你写入的顺序打印
#打印出 {'hello', 6.6, 7}
```

* 词典(一组使用大括号包裹的**键值对**) **Dictionary** :
* 键(key)：用作索引用来查找数据；不可变，不可重复；
* 值(value)：存放的数据，格式无限制；

```
my_dictionary = {'name':'Colored Glaze','age':'19'}
print(my_dictionary) #打印出 {'name': 'Colored Glaze', 'age': '19'}
```

* 元组(一个不可变的有序集合，用小括号包裹) **Tuple** ：

```
my_tuple = (6.6,'hello',7)
print(my_tuple) #打印出 (6.6, 'hello', 7)
```

* 范围(一串数字，常用于循环中) Range ：

```
my_range = range(6)
print(my_range) #只填了一个数字的话就是默认从0到你填写的那个数字
#打印出 range(0, 6)
```

* 列表(支持不同数据的有序元素集合,使用方括号包裹) **list** ：

```
my_list = [6.6,'hello',7,True]
print(my_list) #打印出 [6.6, 'hello', 7, True]
```

* 无(表示值缺失的特殊值) **None** ：

```
my_none = None
print(my_none) #打印出 None
```
