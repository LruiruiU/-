# 打印(print)是怎么用的

* 每一种编程语言都有一种方式通过内置的方法、函数、属性或关键词向终端输出数据;
* 在Python中这个方法就是 **print** 函数，你可以使用这个函数将数据打印到终端，接下来让我们看看这个函数是怎么使用的吧~;

1. 学习任何编程语言时，老师和网上的教学都会让你写一个简单的程序，那就是 **Hell Word！**很积极向上对不对

```
#在Python中只需要这样写
print('Hell Word!')
#就会在终端里打印出 Hell Word! 啦
#注意要使用英文的分号
#不能使用中文的分号
```

1. 在上方的实例中，字符串是传递给函数的参数，print 是可以一次打印多个值或者参数的，只需要用 **逗号(,)** 分隔开就好啦，

```
#就比如这样：
print('我的名字','叫做','琉璃') #是的，里面可以用中文，反正这个只是把你输进去的原样输出，写什么文都行
#当你用逗号把它们分隔开后打印，Python会在打印出的字符串中间自动添加空格（注意：要使用英文的逗号）
#这样就会输出
#我的名字 叫做 琉璃
```
