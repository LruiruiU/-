# 符串连接和字符串插值

* 在处理字符串时。将不同文本片段组合到一起是经常会做的事
* 在Python中，可以使用加号 + 将多个字符串组合到一起
* 这一过程被称为**字符串连接**：

```
str_1 = 'Hello'
str_2 = 'World'

str_plus = str_1 +' '+ str_2
print(str_plus) #输出 Hello World
```

* 要注意，这种拼接方法**只适用于字符串**，如果你想用这种方式将字符串和其它类型的数据拼接起来，那么你将喜提一个大大的报错

```
name = 'Colored_Glaze'
age = 19

name_and_age = name +' '+ age
print(name_and_age)
#TypeError: can only concatenate str (not "int") to str
```

* 这是因为Python在连接数据类型(比如整数)时不会将其自动转化为字符串
* Python要求所有元素均是字符串才能将其连接
* 为了解决这个问题，可以使用Python的内置函数将数字转化为字符串，该函数会返回给定数字的字符串形式，不会修改原对象
* str()

```
name = 'Colored_Glaze'
age = 19

name_and_age = name + str(age)
print(name_and_age) #输出 Colored_Glaze19
```

* 除此之外也可以使用**增强赋值运算符(**算术运算符 + 普通赋值**)**进行串接
* 由 += 表示
* 可以在一步内同时完成连接和赋值

```
name = 'Colored_Glaze'
age = 19

name_an_age = name #以名字为开始
name_an_age += str(age) #将年龄作为字符串追加到末尾

print(name_an_age) #输出 Colored_Glaze19
```

* 将变量和表达式插入字符串的方式被称为**字符串插值**
* Python中有一个叫做 f-strings 的字符串类别(格式化字符串文字的缩写)
* 它允许你用简洁易读的语法处理插值
* F-strings (以小写或大写f开头)允许你用花括号{}包裹的字段内嵌如变量或表达式：

```
name = 'Colored_Glaze'
age = 19
name_and_age = f'My name is {name} ane I am {age} years old'
print(name_and_age) #输出 My name is Colored_Glaze ane I am 19 years old

num1 = 5
num2 = 10
print(f'The sum of {num1} and {num2} is {num1 + num2}') #输出 The sum of 5 and 10 is 15
```
