# 什么是列表，它如何工作

* 列表数据类型是一组有序的元素，可以由字符串、数字，甚至其它列表组成
* 列表是可变的，采用基于0的下标索引，这意味着列表的第一个元素索引为0
* 以下是基础的列表：

```
colored_glaze = ['琉璃', 'age', 19]
```

* 要访问其中的元素的话可以在序列中引用其索引：

```
colored_glaze = ['琉璃', 'age', 19]
colored_glaze[0] 
---------------------------
#如果你要打印出来的话就这样
colored_glaze = ['琉璃', 'age', 19]
print(colored glaze[0]) #输出 琉璃
```

* 索引也可以是负数
* 负索引用于访问从列表末尾开始的元素：

```
colored_glaze = ['琉璃', 'age', 19]
colored_glaze[-1]
```

* 除了以上创建列表的方式外，还有一种方式

## 使用构造函数 list()

* 该函数用于将可迭代对象转化为列表：

```
name = 'colored glaze'
list(name)
-----------------------
#要打印的话就这样写
name = 'colored glaze'
print(list(name)) #输出 ['c', 'o', 'l', 'o', 'r', 'e', 'd', ' ', 'g', 'l', 'a', 'z', 'e']
```

* 可迭代对象是一种特殊类型的对象，可以逐项遍历循环

## 如果你想获取列表中元素的总数的话，那么可以使用 len()

```
numbers = [1, 2, 3, 4, 5, 1]
len(numbers) 
-----------------------------
numbers = [1, 2, 3, 4, 5, 1]
print(len(numbers)) #输出 6
```

## 如果你想在某个索引处更新某个值的话，那么就可以

```
programming_languages = ['Python', 'Java', 'C++', 'Rust']
programming_languages[0] = 'JavaScript' #将 0 处索引原本的 Python 更改为 JavaScript
print(programming_languages) #输出 ['JavaScript', 'Java', 'C++', 'Rust']
```

* 由于列表是可变的，只要输入有效的索引号就可以更新列表中任何的元素

## 如果你输入了一个不存在的索引号，那么喜提一个索引报错 IndexError

```
programming_languages = ['Python', 'Java', 'C++', 'Rust']
programming_languages[10] = 'JavaScript'

"""

  File "<stdin>", line 2, in <module>

```

## 如果你想从列表中移除某一个元素的话，那么可以使用 del

```
colored_glaze = ['琉璃', 'age', 19, '学习Python']
del colored_glaze[3]
print(colored_glaze) #输出 ['琉璃', 'age', 19]
```

* 有时需要检查某个元素是否存在于列表中

## 这时候就需要用到 in

```
colored_glaze = ['琉璃', 'age', 19, '学习Python']
'学习C++' in colored_glaze    # False
'学习Python' in colored_glaze # True
```

## 有时列表中会嵌套其它列表

```
colored_glaze = ['琉璃', 'age', 19, ['学习Python', '身高200米']]
```

### 访问嵌套列表也需要通过索引

```
colored_glaze = ['琉璃', 'age', 19, ['学习Python', '身高200米']]
colored_glaze[3] # ['学习Python', '身高200米']
```

* 如果要访问嵌套列表中元素的话可以：

```
colored_glaze = ['琉璃', 'age', 19, ['学习Python', '身高200米']]
colored_glaze[3][1] #输出 身高200米
```

* 还有一种常见的列表技术

## 解包值

* 从列表中将值解压出来是一种将列表中的值分配到新变量中的技术

```
colored_glaze = ['琉璃', 19, '学习Python']
name, age, job = colored_glaze

print(name) #输出 琉璃
print(age)  #输出 19
print(job)  #输出 学习Python
```

* 如果你需要将列表内剩余的值收集到一个单词里的话，可以使用 \*

```
colored_glaze = ['琉璃', 19, '学习Python']
name, *rest = colored_glaze

print(name) #输出 琉璃
print(rest)  #输出 [19, '学习Python']
```

* 如果赋值运算符左侧的变量数与列表中的总数不一致的话
* 那么将喜提一个值错误 ValueError

## 最后来看的概念是 切片操作符 [:]

* 与之前在字符串的使用类似

```
colored_glaze = ['琉璃', 19, '学习Python', '身高200米']
colored_glaze[1:3] # [19, '学习Python']
```

* 你还可以使用该操作符指定一个步进区间
* 决定所以之间要跳过多少个：

```
numbers = [1, 2, 3, 4, 5, 6]
numbers[1::2] # [2, 4, 6]
```
