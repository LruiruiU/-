# 循环的作用

## 说到循环，首当其冲的当然就是 for 循环

```
programming_languages = ['Rust', 'Java', 'Python', 'C++']

for language in programming_languages: # for 循环依次将 programming_languages 里的元素取出赋予 language 变量
    print(language) #每次循环都执行打印
"""
输出
Rust
Java
Python
C++
"""
```

* 注意， for 循环是要有缩进的
* 如果没有那么你会得到一个 **缩进错误** **IndentationError** :

```
programming_languages = ['Rust', 'Java', 'Python', 'C++']

for language in programming_languages:
print(language)
"""
Traceback (most recent call last):
  File "<stdin>", line 4, in <module>
IndentationError: expected an indented block after 'for' statement on line 3
"""
```

* for循环也可用来遍历其它可迭代对象
* 比如字符串

```
for char in 'code':
    print(char)
"""
输出
c
o
d
e
"""
```

* for循环也是可以嵌套的：

```
categories = ['Fruit', 'Vegetable']
foods = ['Apple', 'Carrot', 'Banana']

for category in categories:
    for food in foods:
        print(category, food)
"""
输出
Fruit Apple
Fruit Carrot
Fruit Banana
Vegetable Apple
Vegetable Carrot
Vegetable Banana
"""
```

* 外层循环会遍历每一个元素
* 内层循环在此基础在再遍历每一个元素

## 循环当然不止有for，还有 while 循环

* while循环会重复执行一段代码，直到某个条件为假才会停止：

```
secret_number = 3 #设定正确答案为 3 
guess = 0 #初始化猜测值为0

while guess != secret_number: #只要用户输入的数字不等于答案就循环执行
    guess = int(input('Guess the number (1-5): ')) #接接收用户输入的值并转为整数存入 guess 变量
    if guess != secret_number: #输入不等于等于答案就打印下行语句
        print('Wrong! Try again.')

print('You got it!') #输入等于答案就终止循环打印语句
---------------------------------------------------
Guess the number (1-5): 2
Wrong! Try again.
Guess the number (1-5): 1
Wrong! Try again.
Guess the number (1-5): 3
You got it!
```

## Python支持 break 、continue 和 statements

### 先来讲 break

* 该函数用于停止循环的执行

```
names = ['琉璃', '玩偶', 'Cloth Dolls']
for my_names in names:
    if my_names == '琉璃':
        break #终止循环
    print(my_names) #由于被终止所以不会打印
-----------------------------------------
names = ['琉璃', '玩偶', 'Cloth Dolls']
for my_names in names:
    if my_names == '琉璃':
        break #终止循环
print(my_names) #输出 琉璃
```

* 遍历每个字符串并匹配对应的字符串

### 接下来是 continue

* 该语句用于跳过本轮循环剩下的代码
* 直接进入下一轮循环

```
names = ['琉璃', '玩偶', 'Cloth Dolls']
for my_names in names:
    if my_names == '琉璃':
        continue 
    print(my_names)
"""
玩偶
Cloth Dolls
"""
"""
第一轮循环匹配到了 '琉璃' 满足 if 跳过打印命令执行下一轮循环
第二轮循环匹配到 '玩偶' 不满足 if 执行下方打印命令
第三轮循环匹配到 'Cloth Dolls' 不满足 if 执行下方打印命令
"""
```

## for循环和while循环都可以搭配else子句使用

* 只有当循环没有被 break 强制终止时 else 内的代码才会执行

```
words = ['sky', 'apple', 'rhythm', 'fly', 'orange']

for word in words:
    for letter in word:
        if letter.lower() in 'aeiou':
            print(f"'{word}' contains the vowel '{letter}'")
            break
    else:
        print(f"'{word}' has no vowels")
"""
'sky' has no vowels
'apple' contains the vowel 'a'
'rhythm' has no vowels
'fly' has no vowels
'orange' contains the vowel 'o'
"""
```

* 外层的for循环把words中的单词遍历一遍
* 内层for循环把letter中的每个字母遍历一遍
* letter.lower()把所有字母转为小写，统一判断是否匹配 'aeiou' 中的字母

* 内层循环找到匹配到任意一个字母的单词就打印提示并结束循环，因为循环被 break 打断所以 else 不会执行
* 若遍历完单词中的字母后没有匹配到字母，则循环正常结束，触发 else 输出 has no vowels
