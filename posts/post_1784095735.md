# 增强赋值运算如何使用

* 增强赋值运算是**将二元运算和赋值操作合并为一条简写语句**
* JavaScript等语言里也有，区别就是叫法不一样
* 下面有一个示例

```
#这是没使用增强赋值运算：
var = 66
var = var + 99

print(var) #输出 165
---------------------
这是使用增强赋值运算：
var = 66
var += 99

print(var) #输出165
```

* 相信这个很一目了然了
* 增强赋值运算的优势在于能够以更加简洁易读的方式更新变量值
* 这减少了可能因为冗余和拼写错误发生的报错

## 不只加号，其它运算符也可以使用：

```
#减法 -=
var = 99
var -= 66

print(var) #输出33
--------------------
#乘法 *=
var = 66
var *= 99

print(var) #输出 6534
--------------------
#除法 /=
var = 55
var /= 11

print(var) #输出 5.0
---------------------
#向下取整 //=
var = 99
var //= 66

print(var) #输出 1
---------------------
#模运算 %=
var = 99
var %= 66

print(var) #输出 33
---------------------
#幂运算 **=
var = 6
var **= 2

print(var) #输出 36
```

## 也可以运用在字符串上：

```
#拼接字符串 +=
str = '琉'
str += '璃'

print(str) #输出 琉璃
---------------------------
#乘法重复字符串 *=
str = '琉璃'
str *= 3

print(str) #输出 琉璃琉璃琉璃
```

* 其它的用在字符串上就会**报错**了
* 如果你好奇C语言中的 ++ 、-- 或 ++x 之类的写在Python里能不能用
* 答案是 **不能**
* 在Python里这么些的话会**被解释为没有实际增减的双一元运算符**
