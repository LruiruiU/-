# 布尔运算符和短路执行机制的作用

* 如果你的if语句实在很复杂，那么可以使用 **and** **、 or 、 not**

‍我们先来看看 **and**

* and 就是 “和”的意思，如果在if语句中使用and，就要同时满足 条件1 和 条件2 才会判断为真，条件1和条件2其中有任何一个不满足都判断为假：

```
citizen = True
age = 25

if citizen and age >= 18:
    print('你有资格投票') # 因为同时满足了 citizen 和 age 所以判断为真，输出 你有资格投票
else:
    print('你没有资格投票')
---------------------------------------------------------------
citizen = True
age =16

if citizen and age >= 18:
    print('你有资格投票') 
else:
    print('你没有资格投票') #因为只满足了 citizen 没有满足 age 所以判断为假，输出 你没有资格投票
```

然后来看看 **or**

* or 就是“或”的意思，如果在if语句中使用or，条件一和条件二满足其一即可判断为真：

```
age = 19
student = True

if age < 18 or student:
    print('你可以享受优惠') # 虽然没满足 age 但因为满足了 student 所以判断为真，输出 你可以享受优惠
else:
    print('你不可以享受优惠')
```

最后就是 **not**

* not 是 “非”它仅接受一个数值然后反转其布尔值
* 它会把假变成真把真变成假
* not的运算结果只会是真(True)或假(False)

```
‍print(not '') # True, 空字符串是假值
print(not 'Hello') # False, 字符串是真值
print(not 0) # True, 0是假值
print(not 1) # False, 1是真值
print(not False) # True, False是假值
print(not True) # False, True是真值
```
