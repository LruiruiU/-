# 常见的字符串方法

* 方法是一种绑定到特定对象或类上使用的函数
* 必须通过 **对象.方法名()** 或者 **类名.方法名()** 的方式进行调用
* Python中提供了很多内置方法来操作字符串
* 包括但不限于：

1. 返回一个新字符串，将其中所有字符都转为大写 **upper()**

```
str = 'Hello World'
upper_str = str.upper()

print(upper_str) # 输出 HELLO WORLD
```

1. 返回一个新字符串，将其中所有字符都转为小写 **lower()**

```
str = 'Hello World'
lower_str = str.lower()

print(lower_str) #输出 hello world
```

1. 返回一个新字符串，去掉指定前置和后尾字符 **strip()**
2. 如果没有传递参数则会移除前后空格

```
str = ' Hello World '
strip_str = tsr.strip()

print(strip_str)
------------------------
str = ' Hello World '
strip_str = str.strip()
ssr =  strip_str.strip('Hd')

print(ssr) #输出 ello Worl
```

1. 返回一个新字符串，所有你指定的字符串中的字符都被替换为你所指定的新字母 **replace( , )**

```
str = 'Hello World'
replace_str = str.replace('Hello','Hi')

print(replace_str) #输出 Hi World
```

1. 将字符串按指定分隔符切开并返回一个列表 **split()**
2. 如果没有指定分隔符则在空白处切开

```
str = 'Hello World'
split_str = str.split()

print(split_str) #输出 ['Hello', 'World']
-----------------------------------------
str = 'Hello/World'
split_str = str.split('/')

print(split_str) #输出 ['Hello', 'World']
```

1. 把列表中零散的字符拼接成一串完整的字符串 **join()**

```
list = ['Hello','World']
join_list = ' '.join(list)

print(join_list) #输出 Hello World
----------------------------------
list = ['Hello','World']
join_list = ','.join(list)

print(join_list) #输出 Hello,World
```

1. 返回一个布尔值表示该字符串是否以指定字符开头 **startswith()**

```
str = 'Hello World'
startswith_str = str.startswith('Hello')

print(startswith_str) #输出 Ture
----------------------------------------
str = 'Hello World'
startswith_str = str.startswith('hello')

print(startswith_str) #输出 False
```

1. 查找字符串中的指定字符，返回它首个字符出现的索引位置 **find()**
2. 找不到就返回 -1

```
str = 'Hello World'
find_str = str.find('World')

print(find_str) #输出 6
----------------------------
str = 'Hello World'
find_str = str.find('world')

print(find_str) #输出 -1
```

1. 返回字符在字符串中出现的次数 **count()**

```
str = 'Hello World'
count_str = str.count('l')

print(count_str) #输出 3
```

1. 返回一个新字符串，第一个字符大写，其他字符小写 **capitalize()**

```
str = 'hello World'
capitalize_str = str.capitalize()

print(capitalize_str) #输出 Hello world
```

1. 如果字符串中所有的字符都是大写则返回 True 若不是则返回 False; **isupper()**

```
str = 'HELLO WORLD'
isupper_str = str.isupper()

print(isupper_str) #输出 Ture
-----------------------------
str = 'HELLO WOrLD'
isupper_str = str.isupper()

print(isupper_str) #输出 False
```

1. 如果字符串中所有的字符都是小写则返回 True 若不是则返回 False; **islower()**

```
str = 'hello world'
islower_str = str.islower()

print(islower_str) #输出Ture
----------------------------
str = 'Hello world'
islower_str = str.islower()

print(islower_str) #输出False
```

1. 返回一个新的字符串，每个单词的首转为字母大写 **title()**

```
str = 'hello world'
title_str = str.title()

print(title_str) #输出 Hello World
```
