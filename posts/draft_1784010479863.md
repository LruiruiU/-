# 整数和浮点数的使用

* 整数和浮点数是Python中常见的数据类型
* 可以使用它们进行数学运算
* 相信都知道什么是整数和浮点数

```
#加法运算 +
int_1 = 56
int_2 = -96.5
ints = int_1 + int_6

print(ints) #输出 -40.5
------------------------
#减法运算 -
int_1 = 79
int_2 = -168.6
ints = int_1 - int_2

print(ints) #输出 247.6
------------------------
#乘法运算 *
int_1 = 9
int_2 = -16.5
ints = int_1 * int_2

print(ints) #输出 -148.5
------------------------
#除法运算 /
int_1 = 982
int_2 = -12.5
ints = int_1 / int_2

print(ints) #输出 -78.56
------------------------
```

## 除了这些最基本的外更复杂的也可以做到

```
#模运算 % 
int_1 = 686
int_2 = -16.5
ints = int_1 % int_2

print(ints) #输出 -7.0
----------------------
#地板除(向下取整) //
int_1 = 535
int_2 = -2.5
ints = int_1 // int_2

print(ints) #输出 -214
----------------------
#指数运算(也就是幂运算) **
int_1 = -5
int_2 = 6.6
ints = int_1 ** int_2

print(ints) #输出 (-12681.901996235616+39030.88099388445j)
------------------------
```

* 有时候，涉及浮点数的运算结果的小数位会超出预期，会带来微小的舍入误差

## 数值类型转换函数

* Python中有内置模块可以将数值或字符串转化为整数或浮点数

```
#整数转浮点数 float()
my_int = 66
my_float = float(my_int)

print(my_float) #输出 66.0
print(type(my_float)) #输出 <class 'float'>
----------------------------
#浮点数转整数 int()
my_int = 66.666
my_float = int(my_int)

print(my_float) #输出 66
print(type(my_float)) #输出 <class 'int'> 
#这个好像不会四舍五入只是单纯的去掉，用下面这个就可以了
----------------------------
#将数字四舍五入到指定小数点位 round()
my_int_1 = 66.666
my_int_2 = 77.777

my_float_1 = round(my_int_1)
my_float_2 = round(my_int_2,1)

print(my_float_1) #输出 67
print(my_float_2) #输出 77.8
#如果没设置就会默认到整数
---------------------------
#返回一个数字的绝对值 abs()
num = -66
abs_num = abs(num)

print(abs_num) #输出 66
---------------------------
#用来计算一个数的次方，也可以带余数快速算乘方取模 pow()
int_1 = pow(3,6)
int_2 = pow(3,6,9)

print(int_1) #输出 729
print(int_2) #输出 0
```
