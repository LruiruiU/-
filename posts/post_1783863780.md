# 什么是字符串切片？它有什么作用

* 在上一篇中学了如何索引来识别字符串，并用方括号表示访问

```
str = "Hello world"

print(str[0])  #输出 H
print(str[6])  #输出 w
print(str[-1]) #输出 d
```

* 字符串切片可以让你提取字符串的一部分，或者只使用字符串的特定部分
* 如果你想从某个索引中提取字符串到另一个索引，只需要使用它们

```
#中间用冒号分隔
string[start:stop]
-------------------------
str = 'Hello world'
print(str[1:4]) #输出 ell
```

* 注意索引是非包含性的，所以只会提取从索引开始到结束(但不包括)索引处的字符
* 你也可以省略开始的索引，这样Python就会默认从0开始提取：

```
str = 'Hello world'
print(str[:7])  #输出 Hello w
```

* 既然可以省略起始点那当然就可以神略结束点
* 这样的话就会默认从起始点开始提取到字符串末尾的所有内容
* 注意，切片不会修改原字符串

```
str = 'Hello world'
print(str[8:])  #输出 rld
```

* 或者你可以直接不要起始和结束
* 这样就会提取整个字符串

```
str = 'Hello world'
print(str)       #输出 Hello world
print(my_str[:]) #输出 Hello world
```

### 除了索引外，还有一个可选参数，用于指定切片中每个索引之间的增量

```
string[start:stop:step]
```

* 在下面的演示中，从0开始提取，到11停止(但是截不到11只能截到10)，步长为2：

```
str = 'Hello world'
print(str[0:11:2]) #输出Hlowrd
```

* 你可以用这个参数做到将字符串反转
* 将步长设为-1，其它留空：

```
str = 'Hello world'
print(str[::-1]) #输出 dlrow olleH
```
