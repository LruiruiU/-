# Python中作用域的作用

* 在Python中，作用域决定了你可以访问变量的节点、
* 它能控制变量的生命周期以及如何在代码的不同部位进行解析
* **局部作用域(L)**：定义在函数或类中的变量
* **外部作用域(E)**：定义在包含或嵌套函数中的变量
* **全局作用域(G)**：在模块或文件顶层定义的变量
* **内置作用域(B)**：Python中预定义函数、模块、关键字和对象的保留名称
* Python使用LEGB规则解析程序中变量的作用域

## 局部作用域意味着在函数或类中声明的变量只能在该函数或类中访问

```
def my_func():
    my_var = 10
    print(my_var)
```

* 这种情况下函数有自己的作用域
* 无法从外部访问，调用时会输出
* 但在函数外部打印将会导致报错：

```
def my_func():
    my_var = 10 #仅在 my_func 中局部作用
    print(my_var)

my_func() # 10

print(my_var) # NameError: name 'my_var' is not defined
"""
my_var 在函数my_func内部赋值，属于局部变量，生命周期、可访问范围仅限my_func函数内部：
进入my_func执行时：系统为my_var分配内存，函数内可以读写该变量，所以内部print(my_var)能打印10；
my_func执行结束后：函数栈销毁，局部变量my_var占用的内存被回收，这个变量就不存在了
"""
```

# 外部作用域意味着一个嵌套在另一个函数之内的函数可以访问其内部所嵌套的函数的变量

```
def outer_func(): #定义了一个函数
    msg = 'Hello there!'

    def inner_func(): #定义了一个函数
        print(msg)

    inner_func()

outer_func() # Hello there!
"""
调用流程：
执行outer_func()，进入外层函数，初始化msg = 'Hello there!'；
定义内部函数inner_func；
在outer_func内部调用inner_func()，内部函数执行print(msg)；
inner_func自身没有定义msg，就向外层outer_func的作用域查找，找到msg的值，打印内容
"""
```

* 要注意一点
* 外部函数无法访问任何嵌套函数中定义的变量：

```
def outer_func():
    msg = 'Hello there!'
    print(res)

    def inner_func():
        res = 'How are you?'
        print(msg)

    inner_func()

outer_func() # NameError: name 'res' is not defined
"""
res是在内层函数inner_func内部定义的局部变量，仅能在inner_func的代码块里访问；外层函数outer_func无法读取内层函数的局部变量，作用域不能逆向穿透
"""
```

* 有一种解决方法
* 在外部作用域中初始化为空字符串，该作用域在内部
* 然后在内部使用关键字 **nonlocal** 创建一个局部变量

```
‍def outer_func():
    msg = 'Hello there!'
    res = ""  # 在外部域中声明 res

    def inner_func():
        nonlocal res  #允许修改一个外部变量
        res = 'How are you?'
        print(msg)  # 从 outer_func() 访问 msg

    inner_func()
    print(res)  # 现在 res 可以被访问和修改

outer_func()
‍
```

## 全局范围指的是那些在程序的任何位置均可被访问的变量

* 这些变量是在任何函数或类之外被声明的
* 这些变量可被任何地方访问
* 甚至包括未被定义于某个函数内的位置

```
my_var = 100 #定义在函数外部，属于全局作用域变量

def show_var():
    print(my_var)

show_var() # 100
print(my_var) # 100
```

* 如果你想让在函数内定义的局部变量可全局访问
* 那么可以使用关键词 **global**

```
my_var_1 = 7

def show_vars():
    global my_var_2
    my_var_2 = 10
    print(my_var_1)
    print(my_var_2)

show_vars() # 7 10

# my_var_2 现在是一个全局变量，可以在程序中的任何位置访问
print(my_var_2) # 10
```

* 也可以使用该关键词修改全局变量

```
my_var = 10  # 一个全局变量

def change_var():
    global my_var  # 允许修改全局变量
    my_var = 20

change_var()

print(my_var)  # my_var 现在已被全局修改为 20
```
