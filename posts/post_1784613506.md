# 什么是元组，元组的作用

* 元组是一种用于创建有序值序列的Python类型
* 元组可以包含混合的数据类型，这点与列表一样

```
colored_glaze = ('琉璃', 19, '学习Python')
```

* 元组和列表是很类似的，使用的方法也大同小异
* 但列表是可变的数据类型，而元组不可变
* 这意味着元组一旦创建则不能更改
* 如果你尝试更改元组中的某一个元素
* 那么你将喜提一个 类型错误 TypeError

## 要访问元组里的元素的话，可以使用方括号包裹索引号

```
colored_glaze = ('琉璃', 19, '学习Python')
colored_glaze[0] # 琉璃
```

### 如果需要访问从元组末尾开始的元素

* 那么就使用负索引

```
colored_glaze = ('琉璃', 19, '学习Python')
colored_glaze[-1] # 学习Python
```

* 如果你尝试寻找一个不存在的索引
* 那么就会 索引错误 IndexError

## 除了常规的创建方法，还有

* 使用 **tuple()**

```
colored_glaze = 'colored glaze'
tuple(colored_glaze) # ('c', 'o', 'l', 'o', 'r', 'e', 'd', ' ', 'g', 'l', 'a', 'z', 'e')
```

* 该方法也可以接受任意的可迭代对象(字符串，列表，甚至其它元组)作为参数，把传入的可迭内容转化为元组

```
#传入字符串：将字符逐个取出构成元组
tuple("abc") # ('a','b','c')
------------------------------
#传入列表：把列表元素转为元组
tuple([1,2,3]) # (1,2,3)
------------------------------
#传入其它元组：复制生成新的元组
tuple((4,5,6)) # (4,5,6)
```

## 如果是要检查某个元素是否属于元组

* 那么可以使用 **in**

```
colored_glaze = ('琉璃', 19, '学习Python'

'学习Python' in colored_glaze     # True
'colored_glaze' in colored_glaze # False
```

## 你也可以像拆列表一样将元组中的元素进行拆包

```
colored_glaze = ('琉璃', 19, '学习Python')
name, age, job = colored_glaze

print(name) #输出 琉璃
print(age)  #输出 19
print(job)  #输出 学习Python
```

* 如果你需要收集元组中所有剩余的元素
* 那么可以使用 \*

```
colored_glaze = ('琉璃', 19, '学习Python')
name, *age = colored_glaze

print(name) #输出 琉璃
print(*age) #输出 19 学习Python
```

## 就像列表一样，你也可以使用切片符提取元组的一部分

```
colored_glaze = ('琉璃', 19, '学习Python', '身高2米2')
colored_glaze[1:3] # (19, '学习Python')
```

* 记住，第一个数字代表起始位置索引
* 第二个数字代表结束位置索引
* 还要注意
* 结束位置索引的元素不包含在被提取的元素中

## 如果你想要移除元组中的一个元素的话

* 那是不肯的
* 最开始就说过了
* 元组不可被更改
* 如果你硬要改的话
* 那么就报错

## 什么时候用得上元组呢

* 如果你需要一个动态的元素集合，可以增删改的话那么就该去使用列表
* 如果你需要的是一个固定不可变的数据集合，那么就该使用元组
