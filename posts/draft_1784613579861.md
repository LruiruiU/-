# 元组的常见用法

## 首先是 count()

* 该函数用于确定一个元素在元组中出现的次数

```
colored_glaze = ('琉璃', 19, '学习Python', '身高2米2', '身高2米2')
colored_glaze.count('身高2米2') # 2
```

* 如果函数中指定的元素不存在
* 那么会返回 0

```
colored_glaze = ('琉璃', 19, '学习Python', '身高2米2')
colored_glaze.count('身高3米3') # 0
```

* 如果函数中没有指定元素
* 那么就会给出一个 类型错误 TypeError

```
colored_glaze = ('琉璃', 19, '学习Python', '身高2米2')
colored_glaze.count()
"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
TypeError: tuple.count() takes exactly one argument (0 given)
"""
```

## 接下来是 index()

* 该函数用于寻找元组中某个元素的索引值

```
colored_glaze = ('琉璃', 19, '学习Python', '身高2米2')
colored_glaze.index(19) # 1
```

* 如果找不到指定的元素
* 那么就会出现一个 值错误 ValueError

```
colored_glaze = ('琉璃', 19, '学习Python', '身高2米2')
colored_glaze.index(20)
"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
ValueError: tuple.index(x): x not in tuple
"""
```

* 该函数也可以用于传递可选的起始和停止索引的参数

```
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
programming_languages.index('Python', 3) #从索引 3 开始往后检索 # 5
```

* 第一个参数指定要搜索的元素
* 第二个参数表示从几号索引处开始往后寻找
* 当然，你还可以加上第三个参数

```
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python', 'JavaScript', 'Python')
programming_languages.index('Python', 2, 5) # 2
```

* 第三个参数表示在几号索引处停止
* 注意，不包括停止的索引处

## 还有就是 sorted()

* 该函数在列表中已经说过，它也适用于元组
* 该函数用于将元组中的元素按序排好

```
numbers = (13, 2, 78, 3, 45, 67, 18, 7)
sorted(numbers) # [2, 3, 7, 13, 18, 45, 67, 78]
```

* 该函数用在元组上是会创建一个排序后的新元组
* 这与用在列表上会原地排序不返回新列表不同
* 因为元组是不可改变的

### 如果你需要自定义排序的方式，那么可以使用 可选参数

```
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')

sorted(programming_languages, key=len) #按照字符串长度进行排序
# ['C++', 'Rust', 'Java', 'Rust', 'Python', 'Python']
----------------------------------------------------------------------------
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')

sorted(programming_languages, reverse=True) #按照倒序排列
# ['Rust', 'Rust', 'Python', 'Python', 'Java', 'C++']
----------------------------------------------------------------------------
```
