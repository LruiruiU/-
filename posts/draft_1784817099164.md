# 准备工作

```
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple requests

```

* 然后在PyCharm里选输入

```
import requests
#导入模块
```

# 寻找音频文件

* 然后就寻找我们想要的音频了
* 找到想要的音频之后按下键盘上的 F12 按键
* 找到 网络那一栏，网络栏里放着的就是该视频的资源包
* 每个资源包都放着对应的功能
* 音频资源包放音频，评论资源包放评论
* 但资源包这么多，怎么找到我们需要的音频资源包呢
* 在网络栏里上方可以选择“字体”“文档”“媒体”等筛选按钮
* 按道理来说音频应该是在“媒体”里的
* 但B站不一样，它放的很深
* 那到底在哪里呢
* 我们可以点击 Fetch/XHR”按钮
* 然后刷新网页，刷新的目的是让所有的网络资源重新加载一边，防止遗漏
* 刷新之后呢，我们点击大小选项，让资源从大到小排序
* 因为视频和音频是相对较大的，视频资源应该是要比音频资源要大的
* 排序了之后要怎么找呢
* B站有一个特征，那就是资源的后面会有“m4s”的特征
* 我们会发现前面有一些，开头、结尾完全一样的资源
* 这些都是视频资源
* 往下看可以看到一个不一样的，那个就是音频链接了
* 点进去可以找到一个“链接地址”，也可能是叫“请求URL”里面放的就是音频链接啦
