# Python：day1

平台：[www.freecodecamp.org](http://www.freecodecamp.org)

## Python是什么

* Python 是一种通用编程语言，以其简单易用著称。这使得 Python 成为新手学习的不二之选

## Python 能做到什么

* Python 被广泛应用于数据科学和机器学习、网页开发、脚本与自动化、嵌入式系统、物联网等多个领域;
* Python 是目前大多数数据科学家和机器学习工程师主要使用的语言；像**Pandas****(Python 最主流的数据分析库,和Numpy搭配使用)**和**Numpy**这样的库可以让数据分析变得不那么繁琐，而对于像**Tensorflow(****深度学习框架，主打神经网络、深度模型)**和**Scikit-learm(传统机器学习工具库，主打轻量、简单、经典算法)**这样的数据库则让机器学习和与AI模型的合作变得更加便捷;
* 在网页开发中，有着像**Django**、**FastAPI**和**Flask**这样的Python框架可以让开发者只需最少的努力就能构建可扩展且安全的后端系统。许多社交媒体平台如**Instagram**和**Pinterest**的后台大多使用Python编写;
* 网络安全人员和黑客也是用Python编写脚本用于检测恶意软件和其它的漏洞与病毒，构建自动化的安全扫描器用于分析威胁;
* Python在像**树莓派**和**MicroPython**兼容的主板上稳定运行，因此你可以构建各种物联网项目，像智能家居，气象监测等都能够实现;
* Python在**DevOps****(开发运维)**中被广泛用于编写**CI****(Continuous Integration持续集成:代码写完提交到仓库后自动跑校验、编译、单元测试、提前找出bug)****/CD****(Continuous Delivery/Deployment持续交付/部署:测试通过后，自动打包、上传、发布到服务器/设备上，一键上线)****脚本**以及跨开发流程管理基础设施，它也常被用于构建后端服务和内部API;
* 在软件测试中，使用如pytest等Python工具创建可靠的测试套件，而系统管理员则依赖Python进行服务监控、日志管理和系统级任务;
* 最后，Python最大的优势之一就是简单易学，你可以编写简单的脚本帮会你完成重复性任务，比如从电子表格中提取数据、发送邮件以及处理本地机器上的邮件;
* 像**Selenium**和**BeautifulSoup**这样的库也能让网站互动变得轻松，可以抓取公共数据，通过网页界面实行自动化任务，甚至管理项目的云部署;
* Python是一门强大而简单的语言
