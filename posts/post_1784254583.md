# Python中的函数

* 函数是可重复调用的代码，调用时运行
* 许多编程语言都有内置函数
* 就像之前一直用的 **print()**

```
name = input('你叫什么名字?') #用户输入自己的名字并回车
print('嗨', name) #输出 嗨 用户输入的名字
```

* 也有将数字、布尔值、字符串转化为整数的 **int()**

```
print(int(3.14)) # 3
print(int('42')) # 42
print(int(True)) # 1
print(int(False)) # 0
```

## 当然，你也可以自己定义函数

* 为此，你需要调用函数的一个关键字、你想赋予函数的名称、一对括号和一个冒号
* 然后在下面运行代码时就可以调用你定义的函数用来运行代码
* 函数运行的代码也被称为函数的主体 **def**

  
‍

```
def hello():
    print('Hello World')
```

* 要使用的话需要一对括号来调用它

```
hello() #输出 Hello World
```

* 还有一个简单的函数可以将两个数字的和输出

```
def calculate_sum(a, b):
    print(a + b)
-----------------------
calculate_sum(3, 1) #输出 4
#其它的运算也可以这么弄
```

* 函数中用逗号分隔开的被称为参数，也就是占位变量，作为调用函数时传递值的 槽位
* 如果没有明确使用参数的话，Python会默认返回 None

```
calculate_sum()

# TypeError: calculate_sum() missing 2 required positional arguments: 'a' and 'b'
```

* 而且

```
def calculate_sum(a, b):
    print(a + b)

my_sum = calculate_sum(3, 1) # 4
print(my_sum) #输出 无
```

* print(a + b)只是在控制台输出相加结果，不会把结果传出函数

## 这时候就要用到 return

* return 作用：终止函数执行、把数据返回给调用函数的位置

```
def calculate_sum(a, b):
    return a + b

my_sum = calculate_sum(3, 1)
print(my_sum) #输出 4
```
